# Sesión 1 – ¿Qué es la programación y por qué Java?

## 1. Bienvenida

¡Hola! Bienvenidos a esta primera sesión del curso **Fundamentos de Java**.  
Hoy vamos a descubrir qué es programar, qué es un lenguaje de programación y por qué Java es tan importante.  
Y al final, haremos juntos nuestros primeros programas. ¡Empecemos!

---

## 2. ¿Qué es programar?

Programar es darle instrucciones a una computadora para que haga lo que queremos.  
Es como explicarle paso a paso cómo resolver un problema o realizar una tarea.

Cada vez que usamos una app, un videojuego o una página web, hay un programador detrás que escribió el código para que todo funcione.  
Podríamos decir que programar es **transformar ideas en acciones** que una computadora puede entender.

---

## 3. Código, programador y sintaxis

En programación hay tres palabras que vamos a escuchar mucho:

- **Código**: son las instrucciones que entiende la computadora.
- **Programador**: es quien analiza el problema y escribe la solución.
- **Sintaxis**: son las reglas del lenguaje, igual que la gramática en un idioma.

Si rompemos esas reglas, el programa no funcionará.  
Aprender la sintaxis es como aprender el **idioma de la máquina**.

---

## 4. ¿Qué es un lenguaje de programación?

Un lenguaje de programación es la forma en que los humanos nos comunicamos con las computadoras.  

Cada lenguaje tiene sus propias palabras y reglas, pero todos sirven para lo mismo: **dar instrucciones claras**.

Por ejemplo, el mismo mensaje se puede escribir en distintos lenguajes.  
En Python se escribe de una manera más corta y en Java un poco más detallada, pero ambos hacen lo mismo: mostrar un texto en pantalla.  
Aunque cambie la forma de escribir, la idea detrás es siempre la misma: decirle a la computadora qué hacer.

---

## 5. ¿Qué es Java?

Java es un lenguaje de programación creado en 1995 por Sun Microsystems.  
Desde entonces, se ha vuelto uno de los más populares del mundo porque es **seguro**, **estable** y funciona en casi cualquier sistema.

Con Java se pueden hacer muchas cosas: páginas web, aplicaciones móviles, videojuegos, programas de empresas y mucho más.  
Por eso, aprender Java es abrir la puerta a muchísimas oportunidades en el mundo de la tecnología.

---

## 6. ¿Por qué Java?

Ahora que ya sabemos qué es Java, hablemos de por qué es tan importante y sigue siendo uno de los lenguajes más usados del mundo.

Java destaca porque combina tres cosas esenciales:

1. **Es seguro y confiable.**  
   Millones de empresas lo usan para manejar información importante sin riesgo de errores graves o ataques.

2. **Es multiplataforma.**  
   Un mismo programa puede funcionar en Windows, macOS, Linux o incluso en celulares Android, sin necesidad de reescribir el código.

3. **Tiene una gran comunidad.**  
   Si tienes un problema, probablemente alguien ya lo resolvió y compartió la respuesta.

Por eso, Java está presente en casi todo: desde aplicaciones bancarias y sistemas universitarios, hasta videojuegos, cajeros automáticos y herramientas de inteligencia artificial.  

Cuando aprendes Java, no solo aprendes a programar: aprendes un lenguaje que te servirá en muchísimos campos de la tecnología.

## 7. JDoodle: el compilador que usaremos

Para este curso vamos a usar **JDoodle** como nuestro compilador de Java.  
JDoodle es una página web que nos permite escribir y ejecutar código **sin instalar nada**, solo usando el navegador.

### 7.1 Primera forma de entrar

1. En el navegador abrimos Google y escribimos `jdoodle`.
2. En los resultados, buscamos la opción que dice **“Online Java Compiler”** y hacemos clic.
3. Esa página nos lleva directo a un editor de Java listo para usar: solo escribimos nuestro código y presionamos **Run** para verlo funcionar.

### 7.2 Segunda forma de entrar

1. En Google escribimos `jdoodle` y esta vez hacemos clic en el resultado **“JDoodle – Free AI powered Online Coding Platform”**.
2. En la página principal, vamos a la barra de búsqueda de lenguajes y escribimos `java`.
3. En la lista que aparece, elegimos **Java**.
4. Se abrirá una ventana para crear un nuevo proyecto:
   - dejamos **Single-file**,
   - usamos la versión de Java que viene por defecto,
   - marcamos **Empty Project** para empezar con un archivo vacío.
5. Hacemos clic en **Start Coding** y listo: ya tenemos nuestro espacio para escribir programas en Java.
