# Fundamentos de Java para Principiantes

## Descripción del curso
Este curso introduce los conceptos básicos de la programación utilizando **Java**, diseñado especialmente para estudiantes de 12 a 17 años sin experiencia previa.  

A través de 5 sesiones breves, los participantes aprenderán a escribir su primer programa, usar variables, condicionales, bucles, métodos y aplicar los principios básicos de la **Programación Orientada a Objetos (POO)**.

---

## Objetivo general
Brindar una introducción práctica y divertida a la lógica de programación con Java, fomentando la resolución de problemas mediante ejercicios guiados y mini proyectos.
